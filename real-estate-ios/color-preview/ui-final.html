SwiftUI 実装プロンプト（全画面分）

以下の確定UIデザインを SwiftUI で実装してください。
アクセントカラー: #007AFF（iOS System Blue）
カード/セクション背景: Color(white: 0.973) ≈ #F8F8FB

===================================
■ 一覧画面（ListingRowView.swift / ListingListView.swift）
===================================
カードスタイル。白背景の上に薄グレーカード。

- ForEach + LazyVStack(spacing: 6) 内に各カード
- カード: HStack(alignment: .top, spacing: 8)
  - VStack(alignment: .leading, spacing: 3):
    - 物件名: .font(.subheadline).fontWeight(.semibold) 1行 truncate
    - メタ行: HStack { 価格(.accentColor.bold) 間取り(.secondary) 面積(.secondary) 徒歩(.secondary) } .font(.footnote)
    - サブ行: "築N年 ・ NF/NF建 ・ 所有権 ・ N戸" .font(.caption2).foregroundStyle(.secondary)
    - 評価行: HStack { 儲かる確率バッジ 沖式時価(.secondary) } .font(.caption2)
  - ハートボタン .font(.callout)
- .padding(11).background(Color.cardBackground).clipShape(RoundedRectangle(cornerRadius: 12))
- ツールバー右: フィルタボタン + ソートボタン

===================================
■ 詳細画面（ListingDetailView.swift）
===================================
グループセクション（Apple Settings 風）。

- ScrollView > VStack(alignment: .leading, spacing: 8):
  - ヘッダー: 物件名(.headline) + 住所(.caption.secondary)
  - メモ: TextEditor .font(.footnote) .background(Color.cardBackground)
  - セクション「物件情報」:
    VStack(spacing: 0) { sectionHeader + detailRow × N }.background(Color.cardBackground).cornerRadius(10)
  - セクション「アクセス・権利」: 同構造
  - 住まいサーフィン評価セクション（アクセントカラー薄背景 + ボーダー）
  - Link ボタン（.borderedProminent / .bordered）
- .padding(.horizontal, 14)

===================================
■ フィルタ画面（ListingFilterSheet.swift）
===================================
アコーディオン折りたたみ式。各セクションのヘッダーをタップで展開/閉じ。

【重要な変更】駅フィルタ → エリア（区）フィルタに変更:
- ListingFilter の stations: Set<String> を wards: Set<String> に変更
- フィルタロジック: listing.address から区名を抽出して比較

構造:
  .presentationDetents([.medium, .large])
  ScrollView {
    VStack(spacing: 0) {
      // 各フィルタセクション: DisclosureGroup スタイル
      FilterAccordion("価格帯", summary: priceSummary) {
        CustomRangeSlider(...)
      }
      FilterAccordion("間取り", summary: layoutSummary) {
        FlowLayout { ForEach(layouts) { FilterChip(...) } }
      }
      FilterAccordion("駅徒歩", summary: walkSummary) {
        Slider(...)
      }
      FilterAccordion("広さ", summary: areaSummary) {
        Slider(...)
      }
      FilterAccordion("権利形態", summary: ownershipSummary) {
        FlowLayout { FilterChip("所有権") FilterChip("定期借地") }
      }
      FilterAccordion("エリア（区）", summary: wardSummary) {
        // エリアごとにグループ化した 3列グリッド
        ForEach(wardGroups) { group in
          Text(group.area).font(.caption2).foregroundStyle(.tertiary)
          LazyVGrid(columns: 3) {
            ForEach(group.wards) { ward in
              WardToggleButton(ward, isSelected: ...)
            }
          }
        }
      }
    }
  }
  .safeAreaInset(edge: .bottom) {
    Button("\(count)件の物件を表示") { dismiss() }
  }

FilterAccordion コンポーネント:
  @State var isExpanded = false
  VStack(spacing: 0) {
    Button { withAnimation { isExpanded.toggle() } } label: {
      HStack {
        Text(title).font(.subheadline).fontWeight(.semibold)
        Spacer()
        Text(summary).font(.footnote).foregroundStyle(.accentColor)
        Image(systemName: "chevron.down")
          .rotationEffect(.degrees(isExpanded ? 180 : 0))
      }
    }.padding(.vertical, 10)
    if isExpanded { content }
    Divider()
  }

区グループデータ:
  let wardGroups = [
